# 🧠 Quiz App (CLI)

A simple command-line quiz application built using Python.
It asks multiple-choice questions and calculates the final score based on user answers.

---

## 🚀 Features

* Multiple-choice questions (MCQs)
* User input-based answers
* Score calculation
* Simple and easy to understand logic

---

## 🛠️ Tech Used

* Python (Core concepts)

  * Loops
  * Dictionary
  * Lists
  * Input/Output

---

## 📂 Project Structure

```
quiz-app/
│── main.py
│── README.md
```

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the script:

```bash
python main.py
```
